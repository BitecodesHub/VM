# Acceptable Use — VM Panel

This is an internal tool. By using it you agree to the following.

- **Authorised use only.** Use the machines for your organisation's work. Do not
  use them for unlawful activity, to attack other systems, to mine cryptocurrency,
  or to host public services.
- **Shared, monitored resources.** Machines run on shared hardware. Administrators
  can see machine ownership, resource usage, and operational logs, and may stop or
  remove machines to protect capacity. Access to the request/access log is limited
  to administrators.
- **No guaranteed persistence.** Treat desktops as working environments, not backup
  storage. Data inside a machine can be lost if the machine is recreated or the host
  fails. Keep anything important in your organisation's normal storage.
- **Idle machines may be stopped** automatically to reclaim capacity; you can start
  them again from the panel.
- **Credentials.** Keep your account password private. Report a lost password to an
  administrator (who can reset it).
- **Data & privacy.** The panel stores your username, a hashed password, session
  metadata (IP, user-agent), and machine ownership. It does not inspect the contents
  of your desktops. Contact an administrator for access or deletion requests.

Administrators may update these terms. Continued use constitutes acceptance.
